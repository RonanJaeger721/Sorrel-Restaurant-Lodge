# Sorrel Restaurant & Lounge

Static website for Sorrel Restaurant & Lounge in Harare.

## What is included

- Responsive single-page restaurant website
- Jamaican and Caribbean menu sections
- Add-to-order cart and pre-order summary
- WhatsApp reservation form
- Quick WhatsApp contact form
- Gallery and location/contact sections

## Run locally

Open `index.html` in a browser.

No build step is required.

## Deploy

This can be deployed on GitHub Pages, Netlify, Vercel, or any static hosting service.
